# 基本的なコマンド

```
git init  <= フォルダ内をgit管理できるように初期化コマンド ★
 => init: initialize(初期化)
git status <= ファイルの変更状況の確認　★
git diff <= ファイルの差分確認 ★
  => diff: diffrent(異なる)
git add <= commit対象に追加　★
git commit <= gitにコミット　★
git checkout <= 差分のあるファイルをもとに戻す・(ブランチの変更) ★
git log <= commitのログを確認する ★

git pull <= リモートから変更差分を取得
git push <= リモートへプッシュ
```
